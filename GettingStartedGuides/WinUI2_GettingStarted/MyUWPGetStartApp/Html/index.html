Click me to go to Bing

You should be able to set focus to the page by clicking and then press any keystroke and the page will display the input. This only works if you have a focusable element on the dom, such as an input text field. You can uncomment one in Html/index.html to see the desired behavior. You can also click the "click me to go to bing" text above and see that input remains broken after navigating to a new page, even if that page has an input field.
